Leap Learn
=========

    Learn gestures with Leap Motion to realize actions. In Javascript.


Installation:
--------------

Set the properties of the Leap Motion device:
> Settings > General > Allow Web Apps    "must be activated".

Run application
--------------------
```sh
node web_server
```
